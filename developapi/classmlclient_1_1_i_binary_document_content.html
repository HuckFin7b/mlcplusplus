Detailed Description

This class is a specialisation of IDocumentContent that holds binary data.

Since
8.0.0

This class provides additional methods to make working with binary data more convenient.

Note
There are also operators like << available for streaming support.
Warning
This class is not yet implemented by a concrete class, or supported by the C++ API. It is here for reference only.
