Configured the CConnection given the specified properties.

See Connection::configure for details.

Note
Assumes digest authentication is being used.
Parameters
connThe CConnection pointer to modify
hostnameThe MarkLogic Server hostname string
portThe MarkLogic Server REST API port as a string
usernameThe connection username (plain text) to use. E.g. admin or someuser
passwordThe connection password (plain text) to use.
usesslWhether to use SSL or not. (I.e. https instead of http)
