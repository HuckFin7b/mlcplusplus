Detailed Description

An ITextDocumentContent instance that wraps a pugixml xml_parse object. (Derived from pugi::xml_document)

Author
Adam Fowler adam..nosp@m.fowl.nosp@m.er@ma.nosp@m.rklo.nosp@m.gic.c.nosp@m.om
Since
8.0.2
Date
2016-07-30

Used by the PugiXmlHelper class

Note
This class has an internal dependency on the pugixml API. As this API is shipped within MarkLogic's C++ wrapper (this API) , this does not introduce any extra dependencies.
