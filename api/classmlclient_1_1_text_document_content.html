Detailed Description

Pure virtual destructor. REQUIRED to allow subclassing.

Pure virtual constructor Sets the textual content for this document

Assumes content string is non null

Parameters
[in]Thestring content to copy in to this object. Returns the content as a string
Returns
The string representation of the content. Returns the number of characters in the content string.
Note
This number does not include C null characters - just std::string length
Returns
The number of characters in the string. Does not include C null character. This class is a specialisation of DocumentContent that holds all data in a string.

This class is used as the data holding class for all JSON and XML documents. There are no JSON or XML specialisations (Use the JSON and XML helper classes in the utilities namespace instead to create, modify, or introspect the JSON/XML.)
